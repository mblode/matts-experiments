# CodePen – Animated Data Table

Animated data table with category toggle and staggered cell animations.

## Pen Details
- **Title:** Animated Data Table (Category Toggle)
- **Description:** Animated data table with category toggle and staggered cell animations.
- **Tags:** react, ui, animation, table, motion

## HTML (CodePen HTML panel)
Paste the contents of `index.html`:

```
<div id="root"></div>
```

## CSS (CodePen CSS panel)
Paste the contents of `index.css`.

## JS / TS (CodePen JS panel)
1. In **Pen Settings → JS → JavaScript Preprocessor**, select **TypeScript**.
2. Add **packages** via **Pen Settings → JS → Add Packages**:
   - `react`
   - `react-dom`
   - `motion`
3. Paste the contents of `index.ts` into the JS panel.

> If CodePen auto‑inserts import lines for the packages, keep them **only once** (no duplicates).

## HTML `<head>` (Pen Settings → HTML → Stuff for <head>)
Add Tailwind browser CDN:

```
<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>
```

## Notes
- CSS preprocessor: **None**
- CSS Base: **Neither**
- Vendor Prefixing: **Neither**
- Requires an internet connection to load the CDN + npm packages.
