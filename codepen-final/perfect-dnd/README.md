# CodePen – Perfect drag and drop

Sortable list with spring physics drag animations

## Pen Details
- **Title:** Perfect drag and drop
- **Description:** Sortable list with spring physics drag animations
- **Tags:** react, ui, animation, drag-and-drop, sortable

## HTML (CodePen HTML panel)
Paste the contents of `index.html`:

<div id="root"></div>

## CSS (CodePen CSS panel)
Paste the contents of `index.css`.

## JS / TS (CodePen JS panel)
1. In **Pen Settings → JS → JavaScript Preprocessor**, select **TypeScript**.
2. Add **packages** via **Pen Settings → JS → Add Packages**:
   - `@dnd-kit/core`
   - `@dnd-kit/sortable`
   - `@dnd-kit/utilities`
   - `clsx`
   - `lucide-react`
   - `mobx-persist-store`
   - `mobx-react-lite`
   - `mobx`
   - `react-dom`
   - `react`
   - `tailwind-merge`
3. Paste the contents of `index.ts` into the JS panel.

> If CodePen auto‑inserts import lines for packages, keep them **only once** (no duplicates).

## HTML <head> (Pen Settings → HTML → Stuff for <head>)
Add Tailwind browser CDN:

<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>

## CSS Settings
- CSS Preprocessor: **None**
- CSS Base: **Neither**
- Vendor Prefixing: **Neither**

## Notes
- Requires an internet connection to load CDN + npm packages.
