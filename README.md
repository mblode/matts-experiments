# Matt's Experiments

A collection of interactive UI/UX experiments built with Next.js, React, and Three.js.

**Live demo:** [experiments.blode.co](https://experiments.blode.co)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the experiments.
